# Reference Answers

這個資料夾保存課程的參考答案、示範提示詞、預期輸出與階段性 UI/UX 實作說明。

## 使用規則

- `Reference_sources/` 放外部資料來源與引用紀錄。
- `Reference_answers/` 放學生任務的參考答案、提示詞、範例截圖與預期成果。
- 參考答案可以幫助教師評分，但不應取代學生自己查證資料與執行 pyCycle 的過程。
- 所有牽涉真實發動機資料的答案，都要回指到 `Reference_sources/source_manifest.md`。
